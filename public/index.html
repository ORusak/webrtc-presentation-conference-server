Product cart api
Application. Release. Tech: nodejs, mongodb, swagger

The goal of the mini-project is to implement a simple "shopping cart" API for an online store for unauthorized users. There is no possibility to register or login to the store. In the store there is an unlimited supply of a fixed set of products (for example, a bottle of milk, a loaf of bread, a sausage stick). There is no user interface for the system. The visitor through the API calls can view the list of products, add any product to his basket any number of times, remove the previously placed product from the shopping cart and view his shopping cart.
